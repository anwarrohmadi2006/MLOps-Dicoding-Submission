# K2 - Membangun Model
**Author: Anwar-Rohmadi**

Folder ini berfokus pada pelatihan model dan pelacakan eksperimen menggunakan MLflow.

## Isi Folder
- `modelling.py`: Script pelatihan model utama dengan integrasi DagsHub.
- `modelling_tuning.py`: Script pencarian hyperparameter (Optuna).
- `model.pkl`: Artifak model terlatih (HistGradientBoostingRegressor).
- `feature_names.json`: Metadata fitur yang dibutuhkan model.
- `metrics.json`: Hasil performa model (RMSE, MAE, R2).
- `screenshots/`: Bukti grafis dashboard DagsHub dan artifak MLflow.
